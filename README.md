# 🏛️ GitHub Domain Intelligence & Generative AI Analytics Platform

An enterprise-tier, full-stack stream analytics dashboard engineered with **Python**, **Streamlit**, and **Seaborn** to ingest, analyze, and visualize high-velocity open-source software repository frameworks. The platform integrates a resilient **Cloud Inference Gateway (DeepSeek LLM)** alongside an automated local synthetic mathematical simulation layer to achieve robust, non-blocking operational continuity under adverse or restricted network conditions.

---

## ⚡ Key Architectural Engineering Pillars

* **9-Variable Statistical Exploratory Data Analysis (EDA):** Implements complex statistical plotting algorithms including multi-variable scatter clusters, kernel density estimations (KDE), linear regressions (`sns.regplot`), range variances (box plots), density distributions (violin plots), and feature correlation heatmaps.
* **Defensive Engineering Resiliency Protocol:** Detects proxy blockages, local firewalls, SSL connection timeouts (`SSLEOFError`), or API rate limits. Automatically switches to a localized numpy-seeded generation layer to safely maintain visual dashboard continuity.
* **Dual-Layer LLM Gateway Integration:** Features a fully sanitized token routing engine designed to process structured repository metadata into strategic markdown reports via DeepSeek cloud inference pipelines, built with an automatic budget/quota fallback safe-handler.
* **Enterprise UI Enhancement Framework:** Utilizes a decoupled custom layout structure containing responsive metrics KPI blocks, responsive navigation sidebars, custom-injected Inter/Segoe UI typography engines, and structured data frames.
